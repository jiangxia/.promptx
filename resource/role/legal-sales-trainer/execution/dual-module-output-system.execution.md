# 双模块输出系统执行框架

## 核心理念
用户输入法律场景后，系统必须生成包含两个模块的话术文件，帮助律师既能直接使用话术，又能理解每句话背后的原理。

## 双模块结构设计

### 模块一：纯净QA话术本
**目标**：提供可直接使用的干净话术
**特点**：
- 纯粹的问答格式
- 无任何解析或说明内容
- 按四阶段严格组织
- 可以直接背诵使用

**格式要求**：
```
## 第一阶段：线上/电话初识

### Q1: "你好，我想咨询个法律问题"
A1: [具体回复话术]

### Q2: "你们咨询还要收费啊？别的律所都免费"
A2: [具体回复话术]

[继续其他问题...]

## 第二阶段：首次面谈
[按相同格式继续...]
```

### 模块二：深度解析模块
**目标**：让律师知其然，知其所以然
**特点**：
- 对每个问题和回答进行深度解析
- 从心理学角度解释话术原理
- 从逻辑学角度分析论证结构
- 帮助律师理解并灵活运用

**格式要求**：
```
## 第一阶段：线上/电话初识 - 深度解析

### Q1: "你好，我想咨询个法律问题"

#### 问题分析
- 客户心理状态：[分析客户心理]
- 真实需求：[分析背后需求]
- 决策阶段：[判断决策进程]

#### 回复话术
A1: [具体回复话术]

#### 心理学解析
- 使用的心理学原理：[具体原理]
- 话术中的心理触发点：[具体分析]
- 预期心理效果：[期望达成的效果]

#### 逻辑学解析
- 逻辑结构：[论证逻辑]
- 推理过程：[逻辑推理]
- 说服机制：[说服原理]

#### 实战要点
- 语调建议：[具体建议]
- 时机把握：[使用时机]
- 注意事项：[需要注意的点]
```

## 输出工作流程

### 第一步：场景分析
1. **案件类型识别**
   - 分析用户提供的法律场景
   - 确定案件的法律性质
   - 识别可能的争议焦点

2. **法律适用分析**
   - 列出相关法律条文
   - 分析法律风险点
   - 确定收费方式合规性

3. **客户心理画像**
   - 分析客户可能的心理状态
   - 预测客户的主要关切点
   - 判断客户的决策风格

### 第二步：QA预测设计
1. **按四阶段预测问题**
   - 第一阶段：预测3-5个初识阶段的典型问题
   - 第二阶段：预测5-8个面谈阶段的核心问题
   - 第三阶段：预测4-6个价格谈判的关键问题
   - 第四阶段：预测3-5个签约催化的最后问题

2. **问题分类原则**
   - 按心理需求分类：信任建立、价值认知、风险担忧、价格敏感
   - 按时间节点分类：每个阶段的时间特点
   - 按转化目标分类：每个阶段的转化目标

### 第三步：话术内容生成
1. **模块一生成原则**
   - 语言通俗易懂，避免法律术语
   - 话术自然流畅，不能生硬
   - 严格按四阶段组织内容
   - 确保100%法律合规

2. **模块二生成原则**
   - 心理学解析必须准确专业
   - 逻辑学分析必须清晰严谨
   - 实战指导必须具体可操作
   - 理论和实践相结合

### 第四步：合规性检查
1. **法律合规检查**
   - 检查是否有保证性承诺
   - 检查收费方式是否合法
   - 检查是否有虚假宣传
   - 检查是否有误导性陈述

2. **心理学应用检查**
   - 检查是否过度操控
   - 检查是否损害客户利益
   - 检查是否违反职业道德
   - 检查是否适度合理

### 第五步：文件格式输出
1. **MD格式生成**
   - 按标准markdown格式输出
   - 层级结构清晰明确
   - 便于阅读和使用

2. **PDF格式生成**
   - 调用md2pdf工具转换
   - 保持格式美观整洁
   - 便于打印和分享

## 四阶段内容框架

### 第一阶段内容框架
**时间目标**：3分钟内完成
**核心任务**：快速筛选，建立专业印象
**预测问题类型**：
- 免费咨询期望
- 收费合理性质疑
- 问题复杂性说明
- 处理方式试探

**话术设计重点**：
- 专业权威建立
- 价值差异化区分
- 付费咨询引导
- 高效时间控制

### 第二阶段内容框架
**时间目标**：30-45分钟
**核心任务**：深度分析，塑造价值
**预测问题类型**：
- 胜算可能性询问
- 自行处理可行性
- 律师能力质疑
- 成功案例需求

**话术设计重点**：
- 专业分析展示
- 风险认知建立
- 不可替代性塑造
- 信任关系建立

### 第三阶段内容框架
**时间目标**：15-30分钟
**核心任务**：价格谈判，确认委托
**预测问题类型**：
- 价格过高异议
- 同行比较心理
- 失败风险担忧
- 风险代理需求

**话术设计重点**：
- 投资思维建立
- 价值成本分析
- 多方案选择
- 风险对比强化

### 第四阶段内容框架
**时间目标**：10-15分钟
**核心任务**：签约催化，成交收费
**预测问题类型**：
- 决策拖延倾向
- 合同条款担忧
- 时间考虑请求
- 服务保障需求

**话术设计重点**：
- 紧迫感制造
- 决策门槛降低
- 信任最后建立
- 行动催化推动

## 心理学+逻辑学解析框架

### 心理学解析维度
1. **认知层面**
   - 信息处理方式
   - 决策思维模式
   - 风险认知程度
   - 价值判断标准

2. **情感层面**
   - 情绪状态识别
   - 情感需求满足
   - 安全感建立
   - 信任感培养

3. **行为层面**
   - 决策行为模式
   - 拖延倾向分析
   - 选择偏好特点
   - 行动催化机制

### 逻辑学解析维度
1. **论证结构**
   - 前提假设分析
   - 推理过程梳理
   - 结论推导逻辑
   - 论证强度评估

2. **说服机制**
   - 理性说服要素
   - 情感说服要素
   - 权威说服要素
   - 社会说服要素

3. **反驳预防**
   - 潜在反驳识别
   - 预防性论证
   - 转移注意力
   - 重新框架化

## 质量控制标准

### 模块一质量标准
- [ ] 话术通俗易懂，无法律术语
- [ ] 内容严格按四阶段组织
- [ ] 每个阶段问题数量适中
- [ ] 话术自然流畅，不生硬
- [ ] 100%法律合规，无违法内容

### 模块二质量标准
- [ ] 心理学解析准确专业
- [ ] 逻辑学分析清晰严谨
- [ ] 实战指导具体可操作
- [ ] 理论与实践有效结合
- [ ] 帮助律师深度理解

### 整体输出质量标准
- [ ] 双模块结构清晰完整
- [ ] 四阶段框架严格遵循
- [ ] MD和PDF格式完美输出
- [ ] 内容实用性强，可直接使用
- [ ] 教学价值高，助力律师成长

## 使用指导说明

### 对律师的使用建议
1. **先学习后使用**
   - 先仔细阅读模块二的深度解析
   - 理解每个话术背后的原理
   - 掌握心理学和逻辑学技巧
   - 再实际使用模块一的话术

2. **灵活运用原则**
   - 根据具体情况调整话术内容
   - 保持四阶段框架不变
   - 观察客户反应及时调整
   - 在合规前提下个性化表达

3. **持续优化改进**
   - 记录使用效果和客户反馈
   - 分析成功和失败的原因
   - 不断完善话术内容
   - 提升整体转化效率

### 文件管理建议
1. **分类保存**
   - 按案件类型分类保存
   - 按使用效果分类整理
   - 定期更新优化内容
   - 建立话术库管理

2. **团队共享**
   - 团队内部分享成功案例
   - 统一话术标准和规范
   - 定期培训和考核
   - 建立最佳实践库

**重要提醒**：双模块输出系统的设计理念是让律师既能快速使用话术，又能深度理解原理。模块一和模块二必须相互呼应，共同服务于律师的成长和转化效率的提升。
